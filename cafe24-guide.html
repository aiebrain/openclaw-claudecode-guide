🛒 Cafe24 자사몰 셋팅 완전 가이드

이 가이드는?

Cafe24에서 처음 자사몰을 만드는 초보자를 위한 A to Z 셋팅 가이드입니다.
회원가입부터 주문관리까지, 실제 운영에 필요한 모든 단계를 다룹니다.

1. 회원가입 및 쇼핑몰 개설

1-1. Cafe24 회원가입

  1. cafe24.com 접속
  2. 우측 상단 "회원가입" 클릭
  3. 사업자 유형 선택
  4. 개인 사업자 (사업자등록증 있는 경우)
  5. 개인 (사업자등록증 없이 시작 — 나중에 전환 가능)
  6. 본인 인증 (휴대폰 또는 아이핀)
  7. 기본 정보 입력 (이름, 이메일, 비밀번호)
사업자등록증 팁

처음에는 개인으로 시작해도 됩니다. 단, PG사 연동 및 정식 결제를 받으려면 사업자등록증이 필수이므로, 본격 판매 전에 세무서에서 발급받으세요.
- 온라인: 홈택스 → 사업자등록 신청
- 업태: 소매업 / 종목: 전자상거래

1-2. 쇼핑몰 개설

  1. 로그인 후 "쇼핑몰 만들기" 클릭
  2. 쇼핑몰 아이디 설정 (영문, 이후 기본 도메인이 됨: 아이디.cafe24.com)
  3. 쇼핑몰 이름 입력 (브랜드명)
  4. 판매 상품 카테고리 선택 (패션/뷰티/식품 등)
  5. 요금제 선택
요금제 월 비용 상품 수 특징
무료(Starter) 0원 100개 기본 기능, 시작용
Standard 약 22,000원 5,000개 대부분의 기능 포함
Professional 약 33,000원 50,000개 고급 기능 + API
Enterprise 별도 문의 무제한 대규모 쇼핑몰
추천

처음 시작이라면 무료 요금제로 개설 후, 상품이 늘어나면 업그레이드하세요.

  1. 개설 완료 → 관리자 페이지(어드민) 자동 접속

1-3. 관리자 페이지 둘러보기

  • 접속 URL: 아이디.cafe24.com/disp/admin
  • 주요 메뉴 구조:
  • 🏠 대시보드 — 매출/주문 현황 한눈에
  • 🎨 디자인 — 테마, 레이아웃 편집
  • 📦 상품 — 상품 등록/관리
  • 💳 결제 — PG사, 결제수단
  • 🚚 배송 — 배송 정책, 택배사
  • 📊 마케팅 — SEO, 쿠폰, 프로모션
  • 📋 주문 — 주문 처리, CS

2. 디자인 테마 선택 및 커스터마이징

2-1. 테마 선택

  1. 관리자 → 디자인 → 디자인 보관함
  2. "테마 스토어" 에서 원하는 테마 검색
  3. 테마 유형:
  4. 무료 테마: 기본 디자인, 커스터마이징 가능
  5. 유료 테마: 3만~30만원, 전문적인 디자인
  6. 업종에 맞는 테마 선택 후 "적용하기"
테마 선택 기준

- 반응형(모바일 대응) 필수 — 쇼핑몰 트래픽의 70% 이상이 모바일
- 판매 상품과 어울리는 레이아웃 (그리드형 vs 리스트형)
- 후기/리뷰 영역이 잘 보이는 테마가 전환율에 유리

2-2. 기본 커스터마이징

  1. 디자인 → 디자인 편집 접속 (스마트 디자인 에디터)
  2. 수정 가능한 영역:
  3. 헤더: 로고, 메뉴, 검색바
  4. 메인 페이지: 배너, 추천 상품, 카테고리
  5. 상품 목록/상세: 레이아웃, 옵션 표시
  6. 푸터: 회사 정보, 고객센터, SNS 링크

2-3. 로고 & 파비콘 등록

  1. 디자인 → 기본 설정
  2. 로고 이미지 업로드 (권장: PNG, 투명 배경, 가로 200~400px)
  3. 파비콘 업로드 (16x16 또는 32x32px, .ico 또는 .png)

2-4. 필수 페이지 설정

관리자 → 쇼핑몰 설정 → 기본 정보에서 아래 내용 입력:

  • 회사명 (상호)
  • 대표자명
  • 사업자등록번호
  • 통신판매업 신고번호 (구청에서 신고 후 발급)
  • 고객센터 연락처 (전화, 이메일)
  • 영업시간
  • 반품/교환 주소
법적 필수사항

통신판매업 신고 없이 영업하면 과태료가 부과됩니다.
- 신고처: 관할 구청 또는 정부24
- 필요 서류: 사업자등록증, 대표 신분증
- 처리 기간: 약 3~5일

3. 상품 등록

3-1. 카테고리 설정

  1. 관리자 → 상품 → 카테고리 관리
  2. 대분류 생성 (예: 의류, 악세서리, 홈리빙)
  3. 중분류/소분류 추가 (예: 의류 → 상의 → 반팔 티셔츠)
  4. 카테고리별 표시 순서 설정
카테고리 설계 팁

- 3단계 이내로 구성 (대 → 중 → 소)
- 고객 관점에서 직관적으로 (내부 분류 기준 ❌, 고객 탐색 기준 ✅)
- 상품이 10개 미만이면 1~2단계로 충분

3-2. 상품 등록 (개별)

  1. 상품 → 상품 등록 클릭
  2. 입력 항목:

기본 정보

항목 설명 팁
상품명 검색에 잘 걸리는 이름 브랜드명 + 상품 특성 + 키워드
상품코드 내부 관리용 SKU 체계적으로 (예: TS-001)
카테고리 분류 선택 복수 카테고리 가능
판매가 고객에게 보이는 가격 —
공급가 원가 (관리용) 비공개
재고 재고 수량 0이면 품절 표시

상품 이미지

  • 대표 이미지: 1장 필수 (권장 1000x1000px 이상, 정사각형)
  • 추가 이미지: 3~8장 권장 (다양한 각도, 착용샷, 디테일)
  • 이미지 형식: JPG 또는 PNG, 파일당 10MB 이하
상품 사진 촬영 팁

- 자연광 또는 소프트박스 조명 사용
- 배경은 흰색 또는 단색이 깔끔
- 스마트폰으로도 충분 — 삼각대 + 자연광이 핵심
- 무료 보정 도구: Canva, Remove.bg (배경 제거)

상품 설명 (상세페이지)

  • 간략 설명: 검색 결과에 표시되는 1~2줄 요약
  • 상세 설명: HTML 에디터로 작성하는 상세페이지
  • 상품 특징, 소재, 사이즈, 사용법
  • 고화질 이미지 + 텍스트 조합
  • 모바일 가독성 고려 (좌우 여백, 글자 크기)

옵션/변형

  • 옵션 설정: 색상, 사이즈 등 (관리자 → 상품 옵션)
  • 옵션별 추가 금액 설정 가능
  • 옵션별 재고 관리 가능

3-3. 대량 등록

상품이 많을 경우:

  1. 상품 → 상품 일괄 등록
  2. 엑셀 양식 다운로드 → 내용 입력 → 업로드
  3. 이미지는 FTP로 별도 업로드 후 경로 매핑

4. 결제 수단 설정 (PG사 연동)

4-1. PG사란?

Payment Gateway — 온라인 결제를 중개하는 서비스입니다. 카드결제, 계좌이체, 간편결제 등을 처리합니다.

4-2. 주요 PG사 비교

PG사 수수료(카드) 특징 추천 대상
토스페이먼츠 약 3.3~3.5% 연동 쉬움, 토스 간편결제 초보 사업자
KG이니시스 약 3.3~3.5% 가장 많이 사용, 안정적 범용
NHN KCP 약 3.3~3.5% 안정적, 다양한 결제수단 범용
나이스페이 약 3.3~3.5% 네이버페이 연동 편리 네이버 연계 시
수수료 참고

수수료율은 매출 규모, 업종, 계약 조건에 따라 다릅니다. 초기에는 대부분 3.3~3.5% 수준이며, 매출 증가 시 협상 가능합니다.

4-3. PG사 심사 전 필수 셋팅 (승인 조건)

PG사 심사 탈락 주의!

PG사는 가맹점 심사 시 쇼핑몰이 실제로 운영 가능한 상태인지 확인합니다.
아래 항목이 미비하면 심사가 반려되거나 지연됩니다.

✅ 서류 준비 (심사 신청 전 필수)

서류 발급처 소요 시간 비고
사업자등록증 홈택스 즉시~1일 간이/일반 모두 가능
통신판매업 신고증 정부24 3~5일 사업자등록 후 신청
대표자 신분증 - - 주민등록증 또는 운전면허증
정산 통장 사본 거래 은행 즉시 사업자 명의 통장
통신판매업 신고 순서

1. 홈택스에서 사업자등록 먼저 완료
2. 정부24에서 통신판매업 신고 (사업자번호 필요)
3. 신고증 발급까지 3~5 영업일 소요
4. 신고증 없이 PG 심사 신청하면 100% 반려

✅ 쇼핑몰 셋팅 (심사 통과 필수 조건)

PG사 심사관이 쇼핑몰에 직접 접속해서 확인합니다. 아래 항목이 없으면 반려됩니다:

1. 상품 등록 (최소 1개 이상)

  • 상품명, 가격, 상품 이미지, 상세 설명이 모두 있어야 함
  • "테스트 상품", "임시" 같은 이름 ❌ → 실제 판매할 상품명으로

2. 사업자 정보 표시 (법적 필수)

  • Cafe24 관리자 → 쇼핑몰 설정 → 기본 정보
  • 상호명 (사업자등록증과 동일)
  • 대표자명
  • 사업자등록번호
  • 통신판매업 신고번호
  • 사업장 주소
  • 대표 전화번호
  • 이메일
사업자 정보 미표시 = 심사 즉시 반려

전자상거래법상 사업자 정보 미표시는 위법입니다.
PG사가 가장 먼저 확인하는 항목이므로 반드시 입력하세요.

3. 이용약관 및 개인정보처리방침 등록

  • Cafe24 관리자 → 쇼핑몰 설정 → 약관 관리
  • 이용약관
  • 개인정보 수집·이용 동의
  • 개인정보처리방침
  • Cafe24에서 기본 템플릿 제공 → 상호명/연락처만 수정하면 OK

4. 배송/교환/반품 정책 명시

  • 상품 상세페이지 하단 또는 별도 페이지에:
  • 배송비 안내 (무료/유료 기준)
  • 배송 소요일
  • 교환/반품 절차 및 비용
  • 환불 정책

5. 도메인 연결 (권장)

  • 기본 도메인(shop이름.cafe24.com)도 심사 가능하지만
  • 독립 도메인 연결 시 승인율 ↑ (신뢰도 상승)

✅ PG사 심사 체크리스트

□ 사업자등록증 발급 완료
□ 통신판매업 신고증 발급 완료
□ 쇼핑몰 상품 최소 1개 등록 (실제 상품명 + 이미지 + 가격)
□ 사업자 정보 모두 입력 (상호/대표자/사업자번호/통판번호/주소/전화)
□ 이용약관 + 개인정보처리방침 등록
□ 배송/교환/반품 정책 명시
□ 정산 통장 사본 준비
□ 대표자 신분증 사본 준비
이 체크리스트를 모두 완료한 후 PG사에 심사 신청하세요!

보통 3~7 영업일 내 승인됩니다. 서류 미비 시 추가 1~2주 소요될 수 있습니다.

4-4. PG사 연동 절차 (심사 후)

  1. PG사 홈페이지에서 가맹점 신청
  2. 토스페이먼츠 → 가맹점 등록
  3. KG이니시스 → 온라인 가맹점 신청
  4. 서류 제출 (위 체크리스트 서류)
  5. 심사 (보통 3~7 영업일)
  6. 승인 후 키(Key) 발급 (상점 ID, API Key 등)
  7. Cafe24 관리자 → 결제 → PG 설정
  8. 사용할 PG사 선택
  9. 발급받은 키 입력
  10. 결제수단 활성화 (신용카드, 계좌이체, 가상계좌, 휴대폰 결제)
  11. 테스트 결제 진행 후 정상 작동 확인

4-5. 간편결제 추가 설정

아래 간편결제는 별도 가맹점 신청이 필요합니다:

  • 네이버페이: 네이버페이 센터 → 가맹점 신청
  • 카카오페이: PG사 통해 연동 또는 직접 신청
  • 토스페이: 토스페이먼츠 연동 시 자동 포함
  • 페이코: 별도 가맹점 신청
초보자 추천 설정

PG사 1개(토스페이먼츠 또는 KG이니시스) + 네이버페이만 있으면 대부분의 결제를 커버할 수 있습니다.

5. 배송 설정

5-1. 배송 정책 설정

관리자 → 배송 → 배송 설정

  1. 기본 배송비 설정
  2. 무료 배송
  3. 고정 배송비 (예: 3,000원)
  4. 조건부 무료 (예: 50,000원 이상 무료, 미만 시 3,000원)
  5. 수량별/무게별 배송비
