💡 코딩 초보자라면 **Claude Pro 구독($20/월)**이 가장 간단합니다.

방법 A — Claude 구독 (추천)

  1. claude.ai 접속 → Sign up
  2. 로그인 후 "Upgrade to Pro" 클릭
  3. 결제 정보 입력 → 구독 시작

방법 B — API Key

  1. console.anthropic.com 접속 → Sign up
  2. 좌측 메뉴 "API keys" → "Create Key"
  3. 키 이름 입력 (예: claude-code-key) → 즉시 복사 저장!
  4. "Billing" → "Add credits" 충전 (최소 $5)

⚠️ API Key는 생성 직후에만 전체 값을 볼 수 있습니다!

📦 Step 1 — Node.js LTS 설치

이미 설치 확인

node --version
  • v24.x.x 출력 → 건너뛰기
  • 오류 → 아래 진행

다운로드 & 설치

  1. nodejs.org 접속
  2. v24.x LTS 다운로드 (.msi)
  3. 설치 마법사 진행:
    • ⚠️ "Add to PATH" 체크 확인 필수!
    • "Tools for Native Modules" → 체크 불필요

설치 확인 (터미널 새로 열기!)

node --version   # → v24.14.0
npm --version    # → 10.x.x

📦 Step 2 — Git for Windows 설치

이미 설치 확인

git --version

다운로드 & 설치

  1. git-scm.com/downloads/win 접속
  2. "64-bit Git for Windows Setup" 다운로드
  3. 설치 시 PATH 설정 주의:
    • ✅ "Git from the command line and also from 3rd-party software" 선택 (기본값)
  4. 나머지 기본값으로 진행

설치 확인

git --version    # → git version 2.47.1.windows.1

📦 Step 3 — Google Antigravity IDE 설치

Antigravity란?

  • Google의 AI 중심 코드 편집기 (VS Code 기반)
  • 무료 퍼블릭 프리뷰
  • Gemini 3 통합, 멀티 에이전트 관리, VS Code 확장 호환

설치

  1. antigravity.google/download 접속
  2. Windows 버전 다운로드 (.exe)
  3. 설치 마법사 진행 → Finish

최초 실행 설정

  1. 테마 선택 (Light/Dark)
  2. 모드 선택 (Agent Mode / Editor Mode — 나중에 전환 가능)

Google 계정 로그인

  1. 상단 계정 아이콘 클릭
  2. "Sign in with Google" → 브라우저에서 로그인
  3. 권한 승인 → 완료

워크스페이스 열기

  1. 프로젝트 폴더 생성 (예: my-first-project)
  2. File → Open Folder → 폴더 선택
  3. "Trust the authors" → Yes

📦 Step 4 — Claude Code CLI 설치

터미널 열기

  • 단축키: Ctrl + `` (백틱)
  • 메뉴: Terminal → New Terminal
  • PowerShell인지 확인 (PS C:\...>)

방법 A — 네이티브 설치 (권장)

1. 실행 정책 확인:

Get-ExecutionPolicy -Scope CurrentUser
  • Restricted이면:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

2. 설치:

irm https://claude.ai/install.ps1 | iex

방법 B — npm 설치 (대안, deprecated 예정)

npm install -g @anthropic-ai/claude-code

설치 확인 (터미널 재시작!)

claude --version    # → claude v1.x.x
claude doctor       # 설치 상태 진단

⚠️ claude 인식 안 될 때

환경 변수 PATH에 수동 추가:

%USERPROFILE%\.local\bin

(시스템 환경 변수 편집 → 사용자 변수 → Path → 새로 만들기)

🔑 Step 5 — Claude Code 인증

방법 A — Claude 구독으로 로그인 (추천)

  1. 터미널에서 claude 입력
  2. 브라우저 자동 오픈 → Claude 계정 로그인
  3. 터미널로 돌아오면 대화 가능

💡 브라우저 안 열리면 c 키 → URL 복사 → 붙여넣기

방법 B — API Key로 인증

[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-xxxxx", "User")

터미널 재시작 후 claude 실행

첫 실행 화면

화면 입력
이용약관 y
모델 선택 Enter (기본값)
프로젝트 신뢰 y

🚀 Claude Code 기본 사용법

대화 모드

claude

프롬프트(>)에 자연어 입력:

> 간단한 웹페이지를 만들어줘
> 이 코드에서 버그를 찾아줘
> README.md 파일을 작성해줘

주요 명령어

명령어 기능
claude 대화 모드 시작
claude "질문" 한 번만 질문
claude doctor 설치 상태 진단
claude --version 버전 확인
/exit 대화 종료
/clear 대화 초기화
/help 도움말

❓ FAQ

Q. 설치 후 claude가 인식 안 됩니다 → 터미널을 닫고 새로 열어보세요. PATH 반영은 새 터미널에서만 됩니다.

Q. Antigravity에서 Claude Code를 어떻게 사용하나요? → Antigravity 내장 터미널(Ctrl + ``)에서 claude` 입력하면 됩니다.

Q. Claude 사용량 제한이 있나요? → Pro: 시간당 제한 있음 / API Key: 잔액 내 무제한
