🔧 STEP 1 — 기존 AI에서 기억 꺼내기

아래 프롬프트를 지금 쓰는 AI 채팅창에 그대로 붙여넣으세요.

지금까지 나와의 대화에서 학습하거나 기억한 내용을 전부 출력해줘.

다음 항목을 빠짐없이 포함해:
- 내 말투, 선호하는 응답 스타일, 포맷 방식
- 내 이름, 직업, 위치, 관심사 등 개인 정보
- 진행 중인 프로젝트, 목표, 자주 다루는 주제
- 내가 자주 쓰는 도구, 언어, 프레임워크
- 내가 수정 요청했던 행동 방식 (항상 해줘/하지 마 등)
- 그 외 학습된 모든 맥락

요약하거나 생략하지 말고 전부 코드블록 하나에 담아줘.
마지막에 빠진 내용이 있는지 확인해줘.
⚠️ ChatGPT 사용자 주의
Settings → Personalization → Memory가 켜져 있어야 기억이 저장됩니다.
꺼져 있었다면 이전할 내용이 없을 수 있어요.

📋 STEP 2 — 새 AI에 기억 심기

3가지 방법 중 상황에 맞게 선택하세요.

방법 A

첫 대화에 바로 붙여넣기 — 가장 간단

새 AI 채팅창을 열고 아래처럼 입력합니다.

이게 내 배경 정보야. 앞으로 대화할 때 참고해줘:

[복사한 내용 붙여넣기]

✅ 바로 시작 가능  |  ⚠️ 새 대화 열 때마다 반복 필요

방법 B

AI별 영구 기억 설정에 저장 — 추천

한 번 저장하면 새 대화를 열어도 자동으로 기억합니다.

AI설정 위치
ChatGPTSettings → Personalization → Custom Instructions
ClaudeSettings → Capabilities → Memory
GeminiGems 만들기 → 시스템 설명란
Perplexity영구 저장 없음 → 방법 C 사용
방법 C

내 컨텍스트 파일 만들기 — 가장 강력

메모장에 아래 형식으로 저장하고, 새 AI 대화 시작 시 파일 첨부.

# 나의 AI 컨텍스트 파일

이름: [이름]
직업: [직업]
관심사: [관심사]

응답 스타일:
- 한국어로 답변
- 핵심 먼저, 간결하게
- 표나 목록 형식 선호

사용 중인 도구: [도구 목록]
진행 중 프로젝트: [프로젝트]

꼭 지켜줄 것:
- 항상: [규칙]
- 절대 하지 마: [규칙]
💡 팁: 파일명을 my-ai-context.txt 로 저장해두면 어떤 AI든 5초 만에 이식 가능합니다.

🔑 Claude 전용 — 공식 Import Memory 기능

Claude Pro 이상 사용자라면 공식 기능을 활용할 수 있습니다.

  1. claude.com/import-memory 접속
  2. 앤트로픽 제공 프롬프트 복사
  3. ChatGPT / Gemini 채팅창에 붙여넣기 → 결과 복사
  4. Claude Settings → Capabilities → Memory에 붙여넣기
  5. 최대 24시간 내 자동 반영
💡 병합 방식: 기존 Claude 메모리를 덮어쓰지 않습니다. 기존 + 새 내용이 합쳐집니다. ✅

반영 안 됐을 때 대처법

  • 텍스트 정제 후 재시도: 핵심 내용만 추려서 다시 Import
  • 수동 입력: Settings → Memory → "Add memory"에서 중요 항목 직접 입력

✅ 실전 체크리스트

  • 기존 AI 메모리 기능 활성화 여부 먼저 확인
  • 추출 프롬프트를 기존 AI에 붙여넣기
  • 출력된 내용 전체 복사
  • 민감한 개인정보 삭제 후 저장
  • 새 AI 영구 설정에 붙여넣기 또는 컨텍스트 파일 생성
  • "내 직업이 뭐야?" 로 이전 확인 ✅
  • 현재 AI 메모리 Export로 백업 (선택)

❓ FAQ

Q. 무료 AI에서도 됩니까?

추출 프롬프트는 어디서나 사용 가능합니다. 단, 영구 저장 기능은 AI마다 유료일 수 있습니다.

Q. 기억을 새 AI에 넣으면 기존 내용이 사라지나요?

아니요. 기존 + 새 내용이 병합됩니다. 덮어쓰기가 아닙니다.

Q. 얼마나 자주 해야 하나요?

AI 교체할 때만 하면 됩니다. 컨텍스트 파일 하나 만들어두면 평생 재사용 가능합니다.

Q. 한국어 프롬프트도 됩니까?

네, 위 가이드의 한국어 프롬프트 그대로 사용하세요.

Q. Claude에서 다른 AI로도 옮길 수 있나요?

Claude에게 "지금까지 기억하는 내용 전부 출력해줘" 요청 후 Export해서 활용 가능합니다.
