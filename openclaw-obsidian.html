🦞 Claude Code × OpenClaw × Obsidian 설치 가이드

완전 초보자를 위한 단계별 안내서 작성일: 2026-03-04 | 소요 시간: 약 30~45분 | 대상: Mac / Windows

📌 개요

도구 설명 비용
Node.js JavaScript 실행 환경. OpenClaw 동작에 필요 무료
Claude Code Anthropic AI 코딩 어시스턴트. 자연어로 코딩 Pro $20/월
OpenClaw 오픈소스 AI 에이전트 플랫폼. 스킬, 멀티 모델, 메시징 연동 무료
Obsidian 마크다운 노트 앱. "제2의 뇌" 무료

설치 순서

🟢 Step 0: Node.js → 🤖 Step 1: Claude Code → 🦞 Step 2: OpenClaw → 💎 Step 3: Obsidian

💡 OpenClaw은 Node.js 22+가 없으면 설치 불가. Claude Code 로그인을 먼저 완료하면 OpenClaw 온보딩이 편리.

🔐 사전 준비

필요 계정

서비스 필요 여부 비용
Anthropic (Claude) 필수 Pro $20/월 또는 Max $100/월
Obsidian 계정 불필요 무료

시스템 요구사항

  • 💻 macOS 12+ 또는 Windows 10/11 (64-bit)
  • 💾 디스크 공간 최소 2GB
  • 🌐 안정적인 인터넷 연결

Anthropic Pro/Max 구독

  1. claude.ai → "Sign Up"
  2. 이메일 인증 → 비밀번호 설정
  3. 프로필 → "Upgrade to Pro" ($20/월 — 워크숍에는 충분)
  4. 카드 정보 입력 → 구독 완료

터미널 여는 법

  • 🍎 Mac: Cmd + Space → "터미널" → Enter
  • 🪟 Windows: 시작 → "Terminal" 검색 (또는 Win + X → PowerShell)

🪟 Windows 사전 설정 (Mac은 건너뛰기)

A1. ExecutionPolicy 변경

# 관리자 PowerShell에서:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Y 입력. 외부 스크립트 실행 허용 (서명된 것만)

A2. 앱 실행 별칭 끄기

  • Win 11: 설정 → 앱 → 고급 앱 설정 → 앱 실행 별칭
  • Win 10: 설정 → 앱 → 앱 실행 별칭
  • node.exe, python.exe → 끄기

A3. Git for Windows 설치

  1. git-scm.com → "Download for Windows"
  2. 기본 설정으로 설치 마법사 진행
  3. 확인: git --version → git version 2.x.x

🟢 Step 0 — Node.js 설치

OpenClaw은 Node.js 22.12.0 이상에서만 동작합니다.

이미 설치 확인

node --version    # v22.12.0 이상이면 건너뛰기

🍎 Mac

방법 1 (추천): nodejs.org → v22.x LTS .pkg 다운로드 → 설치 방법 2: brew install node@22

🪟 Windows

  1. nodejs.org → v22.x LTS .msi 다운로드
  2. 설치 시 체크:
    • ☑ "Automatically install the necessary tools"
    • ☑ "Enable MAX_PATH limit override"

설치 확인 (⚠️ 터미널 새로 열기!)

node --version    # v22.x.x
npm --version     # 10.x.x

🤖 Step 1 — Claude Code 설치

🍎 Mac

# 방법 1 (추천):
curl -fsSL https://claude.ai/install.sh | bash

# 방법 2:
brew install --cask claude-code

🪟 Windows

# 방법 1 (추천):
irm https://claude.ai/install.ps1 | iex

# 방법 2:
winget install Anthropic.ClaudeCode

설치 확인

claude --version    # claude-code x.x.x

로그인 (⚠️ 반드시 여기서 완료!)

  1. claude 입력 → 브라우저 자동 오픈
  2. Claude Pro/Max 계정으로 로그인
  3. 돌아와서 대화 가능 확인

💡 브라우저 안 열리면 c 키 → URL 복사

첫 실행 화면

화면 입력
이용약관 y
모델 선택 Enter
프로젝트 신뢰 y

건강 체크

claude doctor    # 자동 진단

🦞 Step 2 — OpenClaw 설치

Node.js 버전 재확인

node --version    # v22.12.0+ 필수!

🍎 Mac

# 방법 1 (추천):
curl -fsSL https://openclaw.ai/install.sh | bash

# 방법 2:
npm install -g openclaw@latest

🪟 Windows

# 방법 1 (관리자 PowerShell, 추천):
iwr -useb https://openclaw.ai/install.ps1 | iex

# 방법 2:
npm install -g openclaw@latest

온보딩 (초기 설정)

openclaw onboard --install-daemon

온보딩 5단계

단계 설명 입력
1. AI 모델/인증 Anthropic API 키 선택 Claude 계정 사용
2. 워크스페이스 기본 ~/.openclaw/workspace Enter
3. 게이트웨이 기본 포트 18789 Enter
4. 채널 연결 WhatsApp/Telegram/Discord 등 건너뛰기 가능
5. 데몬 설치 백그라운드 자동 실행 진행

💡 초보자는 QuickStart 모드 선택 → 기본값(Enter)으로 빠르게 진행

설치 확인

openclaw gateway status    # → "running"
openclaw doctor            # 전체 진단
openclaw dashboard         # 웹 대시보드 (브라우저)

💎 Step 3 — Obsidian 설치

🍎 Mac

방법 1 (추천): obsidian.md/download → .dmg → Applications로 드래그 방법 2: brew install --cask obsidian

⚠️ "개발자 확인 불가" 경고 → 시스템 설정 → 개인정보 보호 → "그래도 열기"

🪟 Windows

  1. obsidian.md/download → .exe 다운로드
  2. 설치 마법사 진행

보관함(Vault) 만들기

  1. Obsidian 실행 → "새 보관함 만들기"
  2. 이름 입력 (예: "내 노트")
  3. 저장 위치 선택 → "만들기"

🔧 트러블슈팅 / FAQ

Q. command not found 오류

→ 터미널 닫고 새로 열어서 재시도. PATH 반영 문제.

Q. (Win) node 입력 시 Microsoft Store 열림

→ 설정 → 앱 실행 별칭 → Node.js 항목 끄기

Q. (Win) npm install이 10분 이상 걸림

# 관리자 PowerShell:
Add-MpPreference -ExclusionProcess "node.exe"

Q. (Win) 한글 경로 에러 (ENOENT)

mkdir C:\npm-global
npm config set prefix "C:\npm-global"

→ 환경 변수 PATH에 C:\npm-global 추가

Q. 그래도 안 될 때

node --version; claude --version; openclaw doctor

결과 + 빨간색 에러 메시지 전체를 텍스트로 복사해서 공유
